# RNCounter
This is a simple example project about React Native x Redux x Redux Saga (Middleware). The application simple counting navigation UP (increment) and DOWN (decrement).

## Features
- Counter Number
- Up and Down navigation
- Redux
- Redux Saga
- Sample Test
